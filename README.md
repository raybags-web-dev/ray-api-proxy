## simple proxy server

- For now, only get requests are supported
- make calls to any client that accespts a get request

[ ] Query parameters:
```json
base_url,
parameters
```


[ ] Format:

```json
   {
    "base_url": "/proxy-ray-api/base_url?q-parameters",
    "params": "/proxy-ray-api?q=amsterdam",
   } 
```
